# Multiroom chat

Foi criado um sistema de chat que se atualiza em tempo real. Pode ser implementado em outras aplicações, basta se adaptar para isso. 

---

# Tecnologias usadas!

  - Visual Studio Code
  - NodeJS
  -- Express
  -- Express Validator
  -- Body Parser
  -- EJS
  -- Consign
  -- Sockect io
 - Jquery
 - Booststrap
 
---

```sh
$ npm i
```
